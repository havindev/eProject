## 🚀 Cách cài đặt
```bash
# Clone repository
git clone [url-repository]

# Di chuyển vào thư mục
cd eProject

# Cài đặt dependencies (sau khi có code)
composer install
npm install

# Copy file .env
cp .env.example .env

# Generate key
php artisan key:generate

# Migrate database
php artisan migrate

# Run project
php artisan serve
```
